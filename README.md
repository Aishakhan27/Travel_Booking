✈️ Travel Booking Web App

This project is a Travel Booking Web Application developed using Flask and SQLite that allows users to search available trips, select seats, make payments, and view booking history.
The application simulates a real-world travel reservation system with a complete booking workflow and database storage.
🔹 Features
Search available travel routes
Seat selection system
Booking confirmation
Payment simulation
Booking history storage
User-friendly interface using HTML, CSS, and JavaScript
Data stored using SQLite database
🔹 Technologies Used
Python Flask
SQLite
HTML, CSS, JavaScript
Jinja2 Templates
Pandas
🔹 Impact / Learning
Learned full-stack web development using Flask
Implemented database connectivity
Designed complete booking workflow
Improved backend logic and frontend integration
